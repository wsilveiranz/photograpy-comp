#EngCon ANZ Photography Competition requirements

## Users

1. The application will allow anyone from with a Microsoft account to upload load images
2. A group of pre-defined accounts will be tagged as admins.
   
## Adding images

3. A maximum of 5 images per user will be added
4. Images will be tagged agains a user ID but only admins can see the name of the user, so help with anonimization
5. Only image formats will be available to upload
6. Images will be in a vetting area and must be approved by an admin to avoid NSFW images - images are tagged agains a user, but that is a good safety blanket.

## Flow

1. Admins will open a new competition
2. It will be accepting submissions and vetting images.
3. When admins are ready, it will move the competition to voting
4. All approved images will be presenting for voting
5. Other images will be discarded.

## Voting
1. Admins will provide a time window for voting
2. Users will see the anonimized image list and will will have 3 tokens to vote

## Closing competition
1. Competition will close at the end of the window
2. Admins can force close the competition
3. A list of the top 3 images will be presented to admins
4. In case of first image tie, admins will vote to untie
5. Once a clear first place is ready the winner page will present the winner and the two runner ups
6. Winner images will also present the alias of the user - admins will notify users by email out of band.

## Prizes
1. The competition will indicate the prize (or prizes) including description and image

## Site look and feel

The site should have bold large colors, based on the EngCon ANZ branding. it should look festive and modern for user to engage with the competition.

Branding package can be found on the (branding folderr)[/branding].
